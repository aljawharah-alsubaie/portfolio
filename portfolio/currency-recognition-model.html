üéØ Project Overview

Currency recognition remains a fundamental challenge for visually impaired individuals in achieving financial independence. This hybrid machine learning approach combines transfer learning with classical classification methods for Saudi Riyal banknote recognition.

The system utilizes MobileNetV2 as a feature extractor to generate 1280-dimensional feature vectors from currency images, followed by classification using Support Vector Machine (SVM) with Radial Basis Function (RBF) kernel. The dataset comprises 1,256 Saudi banknote images across seven denominations, augmented to 2,447 training samples through systematic geometric and photometric transformations.

Lightweight & Mobile-Ready

The lightweight architecture (~15 MB) enables real-time mobile deployment with expected inference time under 35ms after TensorFlow Lite conversion and INT8 quantization, making it ideal for assistive technology applications.

Transfer Learning

  • 99.67% CV accuracy
  • Minimal overfitting (3.16% gap)
