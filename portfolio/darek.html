üìñ Project Overview

Darek is a user-centric web application designed to revolutionize the property rental experience in Saudi Arabia. The platform provides a unified marketplace where users can discover, evaluate, and book various types of properties including luxury villas, modern apartments, and coastal chalets. The system addresses the fragmented rental market by offering a streamlined, accessible interface for both renters and property owners.

Developed as part of the IT312 Web Applications Engineering course, Darek demonstrates proficiency in modern web development practices including responsive design, client-side interactivity, and accessibility standards. The platform features a complete booking workflow from property discovery through cart management to reservation confirmation, enhanced by promotional offers and community-driven reviews.

Key Technical Features

Dynamic Content Loading: JavaScript-powered content management system that loads property listings, user bookings, and reviews dynamically without page refreshes. Implemented efficient DOM manipulation techniques for optimal performance.

Local Storage Integration: Persistent data management for shopping cart items, user preferences (dark mode), and booking information. Ensures seamless user experience across sessions without requiring backend database.

Responsive Grid Layouts: Mobile-first approach using CSS Grid and Flexbox for adaptive layouts that maintain usability across desktop, tablet, and mobile viewports. Breakpoints optimized for common device sizes.
